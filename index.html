lein-gh-pages

A Leiningen plugin to generate and publish github project pages. This will create a "gh-pages" repository on any project already tracked with git and that has github as the origin. This branch will be empty except for the html files necessary for publishing a github page. It uses http://github.com/quandrum/Pulp as a static site generator

Usage

Put [lein-gh-pages "0.1.0-SNAPSHOT"] into the :plugins vector of your :user profile, or if you are on Leiningen 1.x do lein plugin install lein-gh-pages 0.1.0-SNAPSHOT.

Once a project is in git and github is the designated the origin, you can generate a new project page with the following command:

$ lein gh-pages new

This will create a gh-pages branch and populate it with html files, as well as a templates folder with the base files. You can edit this, and then reupload with

$ lein gh-pages update
